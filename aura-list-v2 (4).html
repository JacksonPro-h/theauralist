How to Use the Admin Panel

🔐 Admin Login

  • 1
    Click ADMIN in the header. Enter the admin password (Jackson09). You'll see an ⚡ Admin badge and the Admin tab appears.

➕ Adding a Level

  • 1
    In the Admin Panel, click + Add Level.
  • 2
    Fill in: Level Name, Creator(s), Verifier, YouTube Video URL (this shows on the level page when clicked), Rank, and Min % to qualify.
  • 3
    Click Save. The level appears immediately. Clicking it in the sidebar shows the embedded video and any accepted records.

📋 Reviewing Submissions

  • 1
    Players submit via the Submit Record page (they must be logged in). Submissions appear under Pending Submissions.
  • 2
    Click Watch video ↗ to verify, then ✓ Accept or ✕ Reject. Accepted records appear instantly on the level's page and count toward scores.

👤 User Accounts

  • !
    Players register via the Sign In / Register button. Accounts are stored in browser localStorage. You can view all registered users and ban them from the Registered Users panel — banned users cannot submit records.

💾 Data

  • !
    All data is in browser localStorage (separate from the first Aura List file). To host publicly, upload this file to GitHub Pages or Netlify.
